JavaScript Resources

There are lots of budding JavaScript programmers out there, and it seems that each and every one of them has a JavaScript page. As you might imagine, the quality ranges from terrific to ... well, you know! Below are some links that I have found useful or at least entertaining. I've tried to divide the sites by category, but there is a lot of overlap and I suggest you explore at random to see what's available.

Tutorials, Documentation, General

  • Introduction to JavaScript provides a JavaScript tutorial with a good number of demonstrations. This site is unique in offering versions in Chinese, Czech, French, German, and Russian. If you can find this site at the above link try its alternate location.
  • The JavaScript 411 site offers a tutorial, FAQ file, a JavaScript "snippet" library, and the oppotunity to upload your own JavaScript examples for inclusion in the library.
  • JavaScript Index is a wide-ranging compendium of links to all kinds of JavaScript-related sites.
  • Netscape's JavaScript resources page includes links to a variety of examples and resources.
  • The Netscape JavaScript Authoring Guide gives you information on JavaScript right from the horse's mouth.
  • Take a gander at the JavaScript listing on Yahoo (a cool search engine - try it out for other topics too!).

Calculators R Us

  • Here's a Monthly Payment Calculator that will let you figure out the monthly payments on that new dual processor Pentium Pro box that you've been drooling over.
  • Calculate how many calories you burn running down to the donut shoppe each morning with the Runner's Calorie Calculator.
  • Finally! Arto's String Calculator provides relief for all of us who need to perform calculations relating musical instrument strings and the notes they produce.
  • A Car Cost Calculator that will confirm that you can't afford it.
  • A basic but well executed Calculator.
  • Compare adjustable rate to conventional mortgages with this Mortgage Scenario Analyzer .
  • Rolf's calculator demonstrates some interesting techniques, including using an image map instead of HTML for the calculator buttons, and using cookies to store the calculator's memory.

Newsgroups

  • comp.lang.javascript A very active group ranging from rank beginners to experienced pros.
  • comp.lang.java Discussion group on JavaScript's "big brother" language.
  • Livesoftware.javascript.developer Another JavaScript newsgroup that tries to concentrate on more advanced topics.
  • Livesoftware - examples This newsgroup is for posting JavaScript snippets and examples.
